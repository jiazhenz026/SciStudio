# E2E Session - ADR-048 PR #1577 Viewer Category Sweep

## 1. Goal And Out-Of-Scope

- **Goal**: Prove the final PR #1577 candidate can route every ADR-048 viewer
  category through `/api/previews/sessions`, including installed-package
  Image/Label previewers discovered through `scistudio.previewers` entry
  points.
- **Out of scope**: Plot editor authoring, MCP plot inspection internals,
  legacy `/api/data/{ref}/preview` compatibility, and visual design review.

## 2. Preconditions

- **Repo state**: `fix/pr1577-adr048-final-blockers` at or after `669ddcf3`
  with the final issue #1644 candidate diff applied.
- **Working tree**: acceptable dirty paths are `.workflow/records/**`,
  `packages/scistudio-blocks-imaging/tests/test_previewer_registration.py`,
  `docs/ai-developer/e2e/2026-06-13-adr048-pr1577-viewers.md`, and
  `docs/planning/adr-048-pr1577-fix-checklist.md`.
- **Worktree to run from**: `.`
- **Backend port**: `8017`
- **Frontend mode**: prebuilt SPA via `scistudio gui`.
- **Required services / env vars**: `SCISTUDIO_DEV` must be unset or set to a
  value other than `1`; `scistudio-blocks-imaging` must be installed from
  package metadata for the current candidate, not discovered only through the
  monorepo fallback.
- **Required data / fixtures**: local files generated under
  `.workflow/local/e2e/adr048-viewers` by Step 1.
- **External accounts**: none.


## 4. Affordances Under Test

- Installed package discovery:
  `importlib.metadata.entry_points(group="scistudio.previewers")`.
- Routed preview session API:
  `POST /api/previews/sessions`.
- Preview resource API:
  `GET /api/previews/sessions/{session_id}/resources/{resource_id}`.
- Package previewer asset API:
  `GET /api/previews/assets/{previewer_id}/viewer.js`.
- Viewer categories:
  DataFrame, Array, Series, Text, Artifact, CompositeData, Collection,
  Image, Label, and PlotArtifact.

## 5. Steps

### Step 1 - Prepare Fixtures And Verify Installed Previewer Entry Point

- **Action**: Run this PowerShell probe from the repository root:
  ```powershell
  $fixtureRoot = Join-Path (Resolve-Path .workflow/local).Path "e2e/adr048-viewers"
  New-Item -ItemType Directory -Force -Path $fixtureRoot | Out-Null
  $env:ADR048_FIXTURE_ROOT = $fixtureRoot
  @'
  import importlib.metadata
  import json
  import os
  from pathlib import Path

  import numpy as np
  import zarr

  root = Path(os.environ["ADR048_FIXTURE_ROOT"])
  root.mkdir(parents=True, exist_ok=True)

  eps = importlib.metadata.entry_points(group="scistudio.previewers")
  previewer_eps = {ep.name: ep.value for ep in eps}
  expected = "scistudio_blocks_imaging.previewers:get_previewers"
  assert previewer_eps.get("imaging") == expected, previewer_eps

  (root / "table.csv").write_text("a,b\n1,2\n3,4\n5,6\n", encoding="utf-8")
  (root / "notes.txt").write_text("hello from adr048 viewer e2e\n", encoding="utf-8")
  (root / "artifact.bin").write_bytes(b"opaque artifact bytes")
  (root / "plot.svg").write_text(
      "<svg xmlns='http://www.w3.org/2000/svg'><rect width='10' height='10'/></svg>",
      encoding="utf-8",
  )
  (root / "composite.zarr").mkdir(exist_ok=True)
  (root / "label.zarr").mkdir(exist_ok=True)

  array = zarr.open(str(root / "array.zarr"), mode="w", shape=(3, 4, 5), dtype="float32")
  array[:] = np.arange(60, dtype=np.float32).reshape(3, 4, 5)
  image = zarr.open(str(root / "image.zarr"), mode="w", shape=(3, 16, 16), dtype="uint16")
  image[:] = np.arange(3 * 16 * 16, dtype=np.uint16).reshape(3, 16, 16)

  manifest = {
      "entry_points": previewer_eps,
      "table_csv": str(root / "table.csv"),
      "array_zarr": str(root / "array.zarr"),
      "image_zarr": str(root / "image.zarr"),
      "label_zarr": str(root / "label.zarr"),
      "text_file": str(root / "notes.txt"),
      "artifact_file": str(root / "artifact.bin"),
      "composite_zarr": str(root / "composite.zarr"),
      "plot_svg": str(root / "plot.svg"),
  }
  (root / "viewer-fixtures.json").write_text(json.dumps(manifest, indent=2), encoding="utf-8")
  print(json.dumps(manifest, indent=2))
  '@ | python -
  ```
- **Expected**: The command exits 0, prints an `imaging` entry point with value
  `scistudio_blocks_imaging.previewers:get_previewers`, and writes
  `.workflow/local/e2e/adr048-viewers/viewer-fixtures.json`.
- **Capture**: console output.
- **On failure**: halt.
